# This is a sample project for creating a video from multiple images

This project is just a sample for imaging how to create a video from multiple images.

> [!NOTE]
> This project does not optimize creating flow then this project gives you the knowledge how to create.

This project contains BGM that is from below Otologic web site.

https://otologic.jp/free/bgm/electronica01.html

---------------------

# 連番画像から動画生成するサンプルプロジェクト

このプロジェクトは Swift を用いて、連番画像から動画を作成するフローをイメージできるサンプルプロジェクトです。

> [!NOTE]
> メモリの最適化などは行っていないので、あくまで各種クラスの利用イメージを掴むためのものです。

内部で利用している動画は以下の Otologic さんの「Specification」を利用させてもらっています。

https://otologic.jp/free/bgm/electronica01.html
